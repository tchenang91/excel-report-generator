# 📑 Excel Report Generator
Automatisierte Erstellung formatierter Excel-Reports aus Datenbank-Abfragen.

## Features
- Formatierte Tabellen mit bedingter Formatierung
- Automatische Diagramme (Balken, Linien, Kreis)
- Mehrere Arbeitsblätter
- Corporate Design (Farben, Schriftarten)

## Verwendung
```bash
python report_gen.py --output monthly_report.xlsx
```
